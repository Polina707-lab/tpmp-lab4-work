Глоссарий содержит основные термины предметной области «Туристическое бюро», используемые в системе.
Маршрут (Route)
Описание пути следования рейса с указанием начального и конечного пунктов.
Рейс (Trip)
Конкретная поездка по маршруту с заданными датами, количеством пассажиров и стоимостью билета.
Автобус (Bus)
Транспортное средство, используемое для выполнения рейсов.
База данных (Database)
Компонент системы для хранения и обработки данных.
Пользователь (User)
Субъект, взаимодействующий с системой с определёнными правами доступа.
Администратор
Пользователь с полным доступом к системе.
Член экипажа
Пользователь с ограниченными правами доступа.
Сессия пользователя (UserSession)
Объект с данными текущего пользователя.
Аутентификация (AuthService)
Процесс проверки логина и пароля пользователя.
Сервис (Service)
Компонент, реализующий бизнес-логику системы.
Отчет
Результат обработки данных системой.
Параметры отчета
Данные, вводимые пользователем для формирования отчета.
Статистика
Результат вычислений на основе данных рейсов.
Стоимость билета
Цена поездки для одного пассажира.
Доход экипажа
Часть дохода от рейса, рассчитываемая системой.
