Описание проекта

Система «Туристическое бюро» — это консольное приложение на языке C++, предназначенное для управления экскурсионными маршрутами, автобусами и экипажами. Приложение позволяет хранить данные о рейсах, рассчитывать статистику и формировать отчёты по выполненным поездкам. Также реализована аутентификация пользователей и разграничение доступа. На сайте представлены основные материалы командного проекта: требования, спецификации, схема базы данных, презентация и ссылка на репозиторий с кодом.
