О проекте

Проект «Туристическое бюро» представляет собой консольное приложение на языке C++, предназначенное для автоматизации учёта экскурсионных маршрутов, автобусов, экипажей и выполненных рейсов.

Приложение поддерживает аутентификацию пользователей, работу с базой данных SQLite, выполнение CRUD-операций и формирование отчётов.

Основные возможности

  • учёт экскурсионных маршрутов;
  • учёт автобусов и экипажей;
  • хранение сведений о выполненных рейсах;
  • формирование отчётов и аналитики;
  • разграничение доступа пользователей.
На сайте представлены основные разделы проекта: документация, спецификации, схема базы данных, презентация, репозиторий и учебные репозитории участников.
