Система «Туристическое бюро» предназначена для управления маршрутами, рейсами, автобусами и экипажами, а также для формирования отчетов.

Акторы

Администратор
Полный доступ к системе.
  • управление рейсами
  • управление автобусами
  • управление маршрутами
  • формирование отчетов
Член экипажа
Ограниченный доступ.
  • просмотр личных данных
  • просмотр начислений
